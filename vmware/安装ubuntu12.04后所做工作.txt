1.sudo apt-get update
2.sudo apt-get upgrade
不要升级Ubuntu版本(do-release-upgrade),可能会出问题。
3.sudo apt-get install vim
4.配置共享服务samba
sudo apt-get install samba samba-common
这里由于是个人使用的虚拟机，不再另外创建samba专用用户，直接使用原有用户
创建好用于共享的目录，我这里为~/work
配置samba
/etc/samba/smb.conf末尾添加：
[work]
     path = home/djf/work
     available = yes
     browseable = yes
；     public = yes
     writable = yes
     valid users = djf
     create mask = 0744
     directory mask =0755
     force user =djf
     force group = djf
     guest ok = no
将security = user的注释去掉
将passdb backend = tbdsam 改成 passdb backend = smbpasswd
将valid users = %S 改为 valid users = djf 并去掉注释

添加samba用户和密码
  sudo touch /etc/samba/smbpasswd
  sudo smbpasswd -a djf
然后按照提示输入密码
重启samba服务：
sudo service smbd restart 
sudo service nmbd restart
或者 
sudo /etc/init.d/smbd restart
